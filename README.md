## Deployment (GitHub Pages)

1. Create a new GitHub repo named `shubhumjosh-eng.github.io` (must be exact)
2. Upload these files to the repository
3. Go to Settings → Pages → Select "main" branch → Save
4. Your site will be live at `https://shubhumjosh-eng.github.io`

Open `index.html` in your browser to preview the portfolio.
